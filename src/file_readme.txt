Titolo del Progetto: Sito Web per uno Studio Legale con Angular

Descrizione del Progetto: Lo scopo di questo progetto è creare un sito web per uno
studio legale utilizzando il framework Angular. Il sito deve avere un design lineare con un
menù contenente quattro sezioni principali: Home, Chi Sono, Contatti e Prenota un
Appuntamento. Il layout del sito deve essere realizzato utilizzando il CSS e deve presentare
colori predominanti di bianco e azzurro (celeste).

Requisiti Specifici:
1. Creare una pagina principale (Home page) che includa una breve introduzione allo
studio legale e fornisca link alle altre sezioni.
2. Creare una pagina "Chi Sono" che contenga informazioni sull'avvocato o lo studio
legale.
3. Creare una pagina "Contatti" che includa informazioni di contatto come indirizzo
email e numero di telefono.
4. Creare una pagina "Prenota un Appuntamento" con un modulo che permetta agli
utenti di richiedere un appuntamento inserendo nome, email, numero di telefono e
una breve descrizione dell'appuntamento richiesto.
5. Creare una "Task List" che visualizzi un elenco di attività (task) da svolgere. Le
attività dovrebbero includere un titolo, una descrizione e uno stato (completato o in
corso).
6. Creare un componente "AddTask" che consenta agli utenti di aggiungere nuove
attività alla Task List. Il componente deve includere campi per il titolo, la descrizione
e la scelta dello stato dell'attività.
